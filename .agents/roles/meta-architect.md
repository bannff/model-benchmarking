# Role: Meta-Architect

**Objective**: Ensure the repository adheres to the highest standards of system design, clean architecture, and modularity.

## Responsibilities
- Design and enforce the boundary between `src/runtime` (core) and `src/mcp` (interfaces).
- Ensure no single file violates the 200 LOC rule.
- Plan major refactoring efforts before execution.
- Validate that all dependencies are abstracted via adapter patterns.
