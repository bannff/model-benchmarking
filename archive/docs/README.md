# Archive Directory

This directory contains archived files from the development process:

## Structure
- `research_files/` - Dataset research and analysis files
- `intermediate_datasets/` - Various dataset processing iterations
- `old_configs/` - Previous training configurations
- `old_scripts/` - Data processing and preparation scripts
- `raw_data/` - Original dataset collections

These files are preserved for reference but are not part of the active training pipeline.
